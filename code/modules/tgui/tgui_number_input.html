/proc/tgui_input_number Creates a TGUI window with a number input. Returns the user's response as num | null.
/proc/tgui_input_real_number A clone of tgui_input_number that defaults to accepting negative inputs too.
/proc/tgui_input_number_async Creates an asynchronous TGUI number input window with an associated callback.
/datum/tgui_input_numberDatum used for instantiating and using a TGUI-controlled number input that prompts the user with a message and has an input for number entry.
/datum/tgui_input_number/asyncAn asynchronous version of tgui_input_number to be used with callbacks instead of waiting on user responses.
